## 🛠️ Getting Started

### 1. Fork the Repository

Click the **Fork** button on GitHub.

### 2. Clone Your Fork

```bash
git clone https://github.com/GodCode7700/background-color-generator-js.git
cd background-color-generator
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

---

## 🌿 Branching Strategy

Use meaningful branch names:

* `feature/add-random-color-history`
* `fix/button-click-bug`
* `style/improve-ui`
* `docs/update-readme`

---

## 🧾 Commit Guidelines

We follow **conventional commits** for clean history.

### Format:

```bash
<type>: <short description>
```

### Types:

* `feat` → New feature
* `fix` → Bug fix
* `style` → UI/design changes
* `refactor` → Code improvement
* `docs` → Documentation

### Examples:

```bash
feat: add random color generator
fix: resolve button click issue
style: improve hover effect
refactor: simplify event logic
docs: update README file
```

---

## 🔄 Pull Request Process

1. Ensure your code works correctly
2. Commit your changes with clear messages
3. Push your branch:

   ```bash
   git push origin feature/your-feature-name
   ```
4. Open a Pull Request (PR)

### PR Checklist:

* ✅ Code is working
* ✅ No unnecessary files
* ✅ Clear description added
* ✅ Follows project structure

---

## 🐞 Reporting Issues

If you find a bug:

1. Go to the **Issues** tab
2. Click **New Issue**
3. Provide:

   * Description of the bug
   * Steps to reproduce
   * Expected behavior
   * Screenshots (if possible)

## 💡 Feature Requests
